

Run the server first, using the *.bat supplied. Alternatively, use...
java Server portNumber inputFile
eg. java Server 3050 "Test Input/c1.dat"

Run the client using the *.bat file supplied. Alternatively use...
java Client
This will start the GUI User interface.

Use the GUI from left to right.
1. Click the connection button - enter the server address and port number
2. Click the Load file button
3. Click the decrypt button

If you wish to change source files on the server, you will need to restart
both the client and server.

If you wish to change source files on the client, you can choose the Load File button
